# MyJourney - منصة سياحية متكاملة لسوريا

## نظرة عامة على المشروع

**MyJourney** هو مشروع منصة سياحية متكاملة ومتطورة تم تطويرها باستخدام أحدث تقنيات الويب لتقديم تجربة سياحية شاملة لاستكشاف جمال سوريا. المنصة تتيح للمستخدمين تصفح الرحلات السياحية، حجز الرحلات، استكشاف المحافظات والأماكن السياحية، وقراءة المقالات والمراجعات.

## الميزات الرئيسية

### 1. إدارة الرحلات السياحية
- عرض شامل للرحلات السياحية مع تفاصيل كاملة (المدة، السعر، عدد الأشخاص، الأماكن المشمولة)
- نظام تصنيف متقدم للرحلات حسب النوع والمحافظة والفئة
- فلترة متقدمة (حسب المحافظة، النوع، السعر، المدة)
- عرض الخرائط التفاعلية لمسار الرحلة
- إدارة العروض الخاصة والرحلات المميزة
- نظام الموافقة والرفض للرحلات المقدمة من المستخدمين

### 2. نظام الحجوزات المتكامل
- حجز الرحلات بسهولة مع إدارة عدد الضيوف
- تتبع حالة الحجز (معلقة، مؤكدة، مرفوضة، ملغاة)
- إدارة الحجوزات من قبل المستخدمين والمسؤولين
- إشعارات تلقائية لتحديثات الحجز

### 3. إدارة المحافظات والأماكن السياحية
- عرض تفصيلي لكل محافظة مع معلومات شاملة
- خرائط تفاعلية للمحافظات والأماكن السياحية
- عرض الأماكن السياحية مع الصور والوصف والإحداثيات
- عرض أفضل أوقات الزيارة لكل مكان
- إحصائيات عن عدد الرحلات والأماكن السياحية

### 4. نظام المقالات والمراجعات
- إنشاء المقالات والمراجعات من قبل المستخدمين
- نظام تقييم للرحلات
- إدارة المقالات من قبل المسؤولين (الموافقة/الرفض)
- عرض المقالات مع إحصائيات المشاهدات

### 5. لوحة تحكم متكاملة
- لوحة تحكم للمستخدمين: إدارة الحجوزات، المقالات، الملف الشخصي
- لوحة تحكم للمسؤولين: إدارة شاملة لجميع جوانب المنصة
- نظام إشعارات متقدم للمستخدمين والمسؤولين
- إحصائيات وتحليلات شاملة

### 6. نظام المصادقة والأمان
- تسجيل دخول وتسجيل حساب جديد
- المصادقة الثنائية (2FA) باستخدام Google Authenticator
- تسجيل الدخول عبر وسائل التواصل الاجتماعي (Socialite)
- نظام إعادة تعيين كلمة المرور
- التحقق من الهوية للمستخدمين

### 7. المميزات التقنية
- واجهة مستخدم حديثة ومتجاوبة باستخدام Tailwind CSS v4
- دعم اللغة العربية والإنجليزية مع إمكانية التبديل بينهما
- خرائط تفاعلية باستخدام Leaflet
- نظام إشعارات بريد إلكتروني متقدم
- نظام إدارة الملفات والصور
- API endpoints للخرائط والحسابات

## التقنيات المستخدمة

### Backend
- **Laravel 12** - أحدث إصدار من إطار عمل Laravel
- **PHP 8.2** - أحدث إصدار من PHP
- **Laravel Fortify** - نظام المصادقة والأمان
- **Laravel Socialite** - تسجيل الدخول عبر وسائل التواصل الاجتماعي
- **Google 2FA** - المصادقة الثنائية

### Frontend
- **Tailwind CSS v4** - إطار عمل CSS حديث
- **Vite** - أداة البناء السريعة
- **Leaflet** - مكتبة الخرائط التفاعلية
- **Axios** - للطلبات غير المتزامنة

### Testing & Quality
- **Pest PHP** - إطار عمل الاختبارات
- **Laravel Pint** - منسق الكود تلقائياً
- **PHPUnit** - اختبارات الوحدة

### Development Tools
- **Laravel Sail** - بيئة التطوير المحلية
- **Laravel Boost** - أدوات التطوير المتقدمة

## البنية التقنية

### نمط التصميم
- **MVC Architecture** - فصل واضح بين المنطق والعرض
- **Repository Pattern** - إدارة البيانات بشكل منظم
- **Service Layer** - فصل منطق الأعمال
- **Form Requests** - التحقق من البيانات بشكل منظم
- **Policies** - إدارة الصلاحيات

### قواعد البيانات
- نظام علاقات متقدم بين الجداول (Eloquent Relationships)
- Migrations منظمة وقابلة للتتبع
- Seeders و Factories للبيانات التجريبية
- نظام فهرسة محسّن للأداء

### الأمان
- حماية CSRF
- التحقق من البيانات (Validation)
- تشفير كلمات المرور
- المصادقة الثنائية
- نظام الصلاحيات والأدوار
